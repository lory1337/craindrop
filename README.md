# craindrop
*Linux console rain drop based on ncurses.*

## requirements
- gcc
- ncurses (libncurses)
- make (optional)
## installation
Compile with make\
`make craindrop`

or with gcc\
`gcc craindrop.c -O craindrop -lncurses`
## usage
`./craindrop`
